#  Spring Boot을 사용한 OAuth2 Google Login



## 요구사항

1. 사용자 인증

* Basic Authentication

* Persistent login

* Federated identity(OAuth2, Google)



2. 유효성 검사

* Server-side

* Client-side



## 사용기술

* Spring Boot

* Spring data JPA(Hibernate)

* MySQL

* Thymeleaf

* Spring Security

* Spring Security OAuth2

* Hibernate Validator

* Jasypt

* ModelMapper
